# Мой Браузер для Xgimi Play 6 (Android TV)

Личный браузер с закладками и категориями. Управление пультом.

## Что умеет
- Плитки-закладки, сгруппированные по категориям
- Добавление / редактирование / удаление закладок (долгое нажатие OK на плитке)
- Открытие сайтов во встроенном браузере (WebView)
- Кнопка «Назад» на пульте работает как в браузере
- Закладки хранятся прямо на проекторе

---

## Как получить готовый APK (без Android Studio)

### Шаг 1. Создай репозиторий на GitHub
1. Зайди на https://github.com → New repository
2. Назови его, например, `tvbrowser`, сделай **Public**, создай.

### Шаг 2. Загрузи эти файлы
- На странице репозитория нажми **uploading an existing file**
- Перетащи ВСЁ содержимое этой папки (включая папки `app`, `.github`, `gradle` и файлы `build.gradle` и т.д.)
- Нажми **Commit changes**

> Важно: папка `.github` должна загрузиться целиком — в ней лежит инструкция для сборки.

### Шаг 3. Дождись сборки
1. Перейди во вкладку **Actions** в репозитории
2. Увидишь запущенный процесс «Build APK» (2–4 минуты)
3. Когда появится зелёная галочка — открой его

### Шаг 4. Скачай APK
- Внизу страницы сборки, в разделе **Artifacts**, будет файл **MyBrowser-APK**
- Скачай и распакуй — внутри `app-debug.apk`

## Как установить на проектор Xgimi Play 6

1. Скопируй `app-debug.apk` на флешку ИЛИ загрузи в облако (Google Drive)
2. На проекторе включи: **Настройки → Безопасность → Установка из неизвестных источников**
3. Открой файл через файловый менеджер (например «X-плеер»/«Send Files to TV»/флешку) и установи
4. Приложение появится в списке приложений как **«Мой Браузер»**

Готово. Добавляй закладки кнопкой «+ Добавить».
